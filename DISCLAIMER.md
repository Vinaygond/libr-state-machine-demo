# Professional Boundary

This repository is a **public technical artifact** maintained as a companion to
Exit Protocol's engineering note on LIBR replay.

## What this repository provides

- A dependency-free Python reference implementation
- Synthetic CSV fixtures
- Regression tests and golden CLI outputs
- A transparent state-machine description of one LIBR calculation path

## What this repository does not provide

- Legal advice
- Expert testimony or expert-certified conclusions
- Court filings or guaranteed admissibility claims
- A substitute for attorney, forensic accountant, or jurisdiction-specific review
- The full Exit Protocol product, including OCR ingestion, evidence vaulting,
  workpaper export, or integrity verification

## Intended audience

- Software engineers evaluating deterministic replay models
- Forensic accountants inspecting calculation mechanics
- Legal technology diligence reviewers examining methodology boundaries
- Researchers looking for edge-case feedback on commingled-account tracing math

## Use responsibly

Outputs from this demo are structured review material for human inspection. Any
real matter requires review of source records, assumptions, jurisdiction, and
professional judgment by qualified counsel or retained experts.
